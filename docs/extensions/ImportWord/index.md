# Import Word

- Import Word Extension for Tiptap Editor.

## Usage

::: code-group

```sh [npm]
npm install mammoth
```

```sh [pnpm]
pnpm install mammoth
```

```sh [yarn]
yarn add mammoth
```

:::

```tsx
import { ImportWord } from '@remarkablejames/rich-text-editor/importword'; // [!code ++]

const extensions = [
  ...,
  // Import Extensions Here
   ImportWord, // [!code ++]
];
```
