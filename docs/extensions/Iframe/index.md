# Iframe

The Iframe extension allows you to add an Iframe to your editor.

## Usage

```tsx
import { Iframe } from '@remarkablejames/rich-text-editor/iframe'; // [!code ++]

const extensions = [
  ...,
  // Import Extensions Here
  Iframe // [!code ++]
];
```
