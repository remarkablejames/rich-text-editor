# Text Bubble

 The Text Bubble extension allows you to show a bubble around the selected text in the editor.

## Usage

```tsx
import { TextBubble } from '@remarkablejames/rich-text-editor/textbubble'; // [!code ++]

const extensions = [
  ...,
  // Import Extensions Here
  TextBubble // [!code ++]
];
```
